# 水下图像增强系统 python182

## 项目概述

水下图像增强系统是基于Python开发的一套图像处理软件，旨在通过先进的图像增强技术改善水下拍摄图像的质量。

## 技术栈

- **核心图像处理**：OpenCV
- **图像格式与显示**：PIL/Pillow
- **图形用户界面**：tkinter

## 功能模块

### 多参数实时调整

- CLAHE对比度限制增强
- 伽马校正调整图像明暗分布
- 亮度、对比度和饱和度调节
- 锐化强度控制

### 核心处理算法

- 自动白平衡校正
- CLAHE算法增强
- 伽马校正优化
- LAB色彩空间处理
- 自适应高斯锐化

### 用户友好界面

- 原始图像与处理结果左右分屏显示
- 参数实时调整滑块控件
- 防抖处理机制
- 自适应图像显示与自动缩放
- 多线程处理防止界面卡顿

### 实用功能

- 支持多种图像格式打开
- 处理结果保存为JPG或PNG格式
- 一键重置参数
- 错误处理与提示

## 系统角色

- **用户**：通过GUI界面导入图片，实时调整图像增强参数，并保存处理结果。
- **系统**：接收用户输入，执行图像处理操作，反馈处理结果。

## 运行环境

- Python 3.x
- 相关库：OpenCV, PIL/Pillow, tkinter

## 部署步骤

1. 安装Python及必要的库
2. 克隆或下载项目代码
3. 运行主程序启动图形界面

## 核心亮点

- **实时处理**：多线程技术保障用户交互流畅性，实时反馈处理效果。
- **算法优化**：结合多种图像处理技术，有效提升水下图像质量。
- **性能优化**：参数防抖机制，减少不必要的计算，提高处理效率。
